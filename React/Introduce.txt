React

    声明式：
        React 的 UI 公式是 UI = f(data); 声明式的开发模式可以在数据改变时更好的反馈在页面上。

    组件化：
        每个页面都由若干个组件组成，易于维护和代码复用。能使复杂的页面简单化；而且组件化也可以更好的传递数据和 Dom 拆分。


    React 提供的 Api 也很简洁，对于组件的状态更新来讲，只是用 setState() 就足够了。每次使用就会触发组件的重新渲染，即使的将数据反馈在页面上。

    React 提供了 JSX 语法，这种语法允许你在 Js 代码中使用 html 标签语法。这使你回到了熟悉的 html 语法中。可以快速的定义需要的 dom 结构。
